Diego Arriagada Zamora, known online as **V0rt3xS0urc3**, is a Chilean pentester and ethical hacker focused on offensive security across infrastructure, web, wireless, and Active Directory environments. His work spans building real-world attack tooling — including a fully Dockerized Kali Linux pentesting station with GPU acceleration — designing enterprise-style network segmentation labs at home, and publishing detailed CTF writeups to document tradecraft and reinforce practical skills. Everything in this portfolio reflects hands-on lab work, not theory.

## Specializations

Diego's offensive security focus is spread across five core disciplines, each backed by practical tooling, real lab environments, and ongoing platform practice.
