## Quick Start: Kali Portable

If you want to get a fully equipped red team workstation running immediately, Kali Portable is the fastest path. The automated installer handles Docker, GPU configuration, image building, and wordlist downloads without any manual steps.

```bash
# 1. Clone the repository
git clone https://github.com/V0rt3xS0urc3/RedTeam-Portfolio.git
cd RedTeam-Portfolio/kali-portable

# 2. Run the automated installer (~30-45 minutes)
chmod +x scripts/install-on-new-pc.sh
./scripts/install-on-new-pc.sh

# 3. Launch the environment
./run-kali.sh normal
```

<Tip>
  Use `./run-kali.sh wpa2` instead of `normal` when you need privileged USB hardware access for WiFi auditing with a monitor-mode-capable adapter.
</Tip>
